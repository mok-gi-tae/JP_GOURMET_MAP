# 기능 명세

## 1. 회원

### 회원가입

유저는 회원가입을 할 수 있다.

### 로그인

유저는 로그인을 할 수 있다.

---

## 2. 지도

### 관광지 조회

사용자는 관광지를 선택할 수 있다.

### 관광지 확대

관광지를 선택하면 주변의 맛집을 볼 수 있다.

---

## 3. 맛집

### 맛집 조회

확대된 관광지 근처 맛집의 상세 정보를 조회할 수 있다.

### 상세 조회

상세 정보에는 
1. 타베로그의 별점 및 URL
2. 맛집 관련 유튜브 영상 URL
3. 맛집이 취급하는 메뉴 종류
를 확인 가능하다.

---

## 4. 리뷰

### 리뷰 작성

사용자는 관련 맛집에 리뷰를 1개만 작성할 수 있다.

### 리뷰 수정

사용자는 자신이 작성한 리뷰를 수정할 수 있다.

### 리뷰 삭제

사용자는 자신이 작성한 리뷰를 삭제할 수 있다.

---

## 5. AI 추천

AI 추천 기능은 사용자가 입력한 관광지, 메뉴 종류, 예산, 혼밥 가능 여부 등의 조건을 바탕으로 DB에 저장된 맛집 후보를 조회하고, 해당 후보들 중 적절한 맛집을 추천한다.

## 6. 기능 우선순위

| 구분 | 기능 | 우선순위 | 비고 |
|---|---|---|---|
| 회원 | 회원가입 | Must | 필수 |
| 회원 | 로그인 | Must | 필수 |
| 지도 | 관광지 조회 | Must | 필수 |
| 지도 | 관광지 확대 | Must | 필수 |
| 맛집 | 맛집 조회 | Must | 필수 |
| 맛집 | 상세 조회 | Must | 필수 |
| 리뷰 | 리뷰 작성 | Should | 가능하면 구현 |
| 리뷰 | 리뷰 수정/삭제 | Should | 가능하면 구현 |
| AI | 맛집 추천 | Must | 포트폴리오 핵심 |
| 즐겨찾기 | 맛집 저장 | Could | 시간이 남으면 |
| 관리자 | 맛집 등록 | Could | 시간이 남으면 |
