# 🇯🇵 JP_GOURMET_MAP

## 📖 Project Introduction

JP_GOURMET_MAP is a web application that helps users discover restaurants around famous tourist attractions in Japan.

Users can:

- View famous tourist attractions on a map
- Find nearby restaurants
- Check Tabelog ratings
- Watch related Korean YouTube videos
- Write reviews and ratings
- Receive AI-based restaurant recommendations
